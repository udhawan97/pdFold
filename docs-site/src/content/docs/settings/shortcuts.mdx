This is a short, honest list — it will grow as more of the app gains dedicated shortcuts.

| Shortcut | Action |
| --- | --- |
| <Kbd>⌘F</Kbd> | Find in workspace |
| <Kbd>⌘Z</Kbd> | Undo |
| <Kbd>⇧⌘Z</Kbd> | Redo |
| <Kbd>⇧⌘O</Kbd> | Open files |
| <Kbd>⇧⌘E</Kbd> | Export |
| <Kbd>Esc</Kbd> | Cancel a dialog |
| <Kbd>Return</Kbd> | Confirm a dialog's default action |

Standard macOS text-editing shortcuts (copy, paste, select all, and so on) work anywhere you're typing in Orifold.
