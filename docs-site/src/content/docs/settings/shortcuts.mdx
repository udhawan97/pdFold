Orifold is a Mac app, and its shortcuts follow familiar Mac conventions — the same
muscle memory you already have from Preview, Word, and Acrobat carries straight over.

Standard macOS text-editing shortcuts (copy, paste, select all, and so on) work
anywhere you're typing in Orifold — in search, comments, form fields, or editable
PDF text.

## Most useful

| Shortcut | Action |
| --- | --- |
| <Kbd label="Command O">⌘O</Kbd> | Open a workspace |
| <Kbd label="Command S">⌘S</Kbd> | Save |
| <Kbd label="Command E">⌘E</Kbd> | Export |
| <Kbd label="Command P">⌘P</Kbd> | Print |
| <Kbd label="Command Z">⌘Z</Kbd> | Undo |
| <Kbd label="Command Y">⌘Y</Kbd> | Redo |
| <Kbd label="Command F">⌘F</Kbd> | Find |

## File

| Shortcut | Action |
| --- | --- |
| <Kbd label="Command O">⌘O</Kbd> | Open a workspace |
| <Kbd label="Command Shift O">⌘⇧O</Kbd> | Add files to this workspace |
| <Kbd label="Command S">⌘S</Kbd> | Save |
| <Kbd label="Command Shift S">⌘⇧S</Kbd> | Duplicate (Save As) |
| <Kbd label="Command E">⌘E</Kbd> | Export |
| <Kbd label="Command P">⌘P</Kbd> | Print |
| <Kbd label="Command Shift P">⌘⇧P</Kbd> | Print 2-up (two pages per sheet) |
| <Kbd label="Command W">⌘W</Kbd> | Close window |

### While editing PDF text

These apply to the text you're editing inline on the page, and to the selection
within it.

| Shortcut | Action |
| --- | --- |
| <Kbd label="Command B">⌘B</Kbd> | Bold |
| <Kbd label="Command I">⌘I</Kbd> | Italic |
| <Kbd label="Command U">⌘U</Kbd> | Underline |
| <Kbd label="Command Shift C">⌘⇧C</Kbd> | Copy style (Format Painter) |
| <Kbd label="Command Shift V">⌘⇧V</Kbd> | Paste style (Format Painter) |
