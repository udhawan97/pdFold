## Steps

1. Open the Document Comfort popover from the annotation toolbar.
2. Enable Night Mode.
3. Choose a warmth preset: **Gentle** (a light warm tint), **Paper** (a cream, paper-like tone), or **Amber** (the warmest, dimmest option for late-night reading).

<Figure alt="PLACEHOLDER for screenshots/night-mode-comparison.png — the same demo page shown three times side by side under Gentle, Paper, and Amber — dark mode UI chrome, 1600×500, 2x" />

## What you should see

The page tint and brightness change immediately; switch presets anytime from the same popover.
