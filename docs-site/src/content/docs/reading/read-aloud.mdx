## Start and control reading

1. Go to the page where reading should begin.
2. Choose **View → Read Aloud** or **More → Read Aloud**.
3. Use the floating control to pause or resume, stop, or change the speech speed.

Orifold follows the spoken passage and advances the canvas when speech crosses onto the next page. Choosing Read Aloud again stops the session.

<Callout variant="note">
	Speech uses the voices built into macOS and document text stays on your Mac. Scanned pages need an OCR text layer before they can be spoken meaningfully.
</Callout>
