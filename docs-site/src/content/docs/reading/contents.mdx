</Callout>

## Open Contents

Choose **View → Contents**, use **More → Contents**, or press <Kbd>⌥⌘1</Kbd>. Expand nested rows and click a title to jump to its destination page.

Orifold reads embedded PDF outlines and creates a hierarchy from Markdown headings during import. Normal PDF export writes the outline back after page assembly.

<Callout variant="note">
	Booklet and N-up output intentionally omits original bookmarks: several source pages can map onto one imposed sheet, so preserving the old destinations would be misleading.
</Callout>
