## Steps

1. Select the highlight, underline, or strikeout tool from the annotation toolbar.
2. Drag across the text you want to mark.
3. Pick a color from the palette — highlight yellow and coral are the defaults.

<Figure alt="PLACEHOLDER for screenshots/annotate-markup-tools.png — annotation toolbar open with highlight/underline/strikeout tools, one highlight already placed on demo text — dark mode, 1600×1000, 2x" caption="Marking up a page from the annotation toolbar." />

## What you should see

The markup appears over the selected text immediately and is listed in the inspector's annotation list.
