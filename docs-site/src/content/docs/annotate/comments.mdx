## Steps

1. Add comments the same way as notes — from the annotation toolbar.
2. Open the inspector to see every comment and annotation across the current document, as a review index.

<Callout variant="tip">
	Hidden Orifold comment metadata is automatically stripped from flat PDF exports — your review chatter never accidentally ships to whoever receives the final file.
</Callout>

## What you should see

The inspector lists every comment with its page location, so you can jump straight to any of them.
