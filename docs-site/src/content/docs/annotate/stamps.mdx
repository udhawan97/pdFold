## Watermarks

1. Choose a watermark from the decorations menu.
2. Adjust its text, opacity, and rotation.
3. Apply it across all pages, or just the current one.

## Page numbers

Add a page-number decoration and choose its position and starting number.

## Bates labels

Bates numbering stamps a sequential identifier (a prefix plus a zero-padded number) across every page in a packet — the format legal exhibits expect.

1. Add a Bates label decoration.
2. Set the prefix (e.g. `EXHIBIT-`) and the padding (e.g. `000001`).
3. The sequence continues across every document in the workspace, in sidebar order.

<Callout variant="note">
	Stamps and decorations are burned into the page at export — they aren't editable after the file leaves Orifold. That makes a Bates-numbered exhibit stable, but it is not a cryptographic authenticity guarantee.
</Callout>

## What you should see

Every exported page carries the watermark, page number, or Bates label you configured, in a fixed position and sequence.
