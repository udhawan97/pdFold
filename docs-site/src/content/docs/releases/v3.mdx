Orifold v0.3 keeps the local-first document workspace from v0.2, adds a supplemental local PDF processing backend, and improves the install lifecycle.

- **Automatic update check on launch** — the Desktop `Orifold.command` launcher checks for updates every time it opens Orifold. See [Update & uninstall](/Orifold/get-started/update-uninstall/).
- **Clean uninstall command** — installs now create `Uninstall Orifold.command` on the Desktop.
- **Local PDF processing backend** — PDF imports flow through an injectable processing engine, with PDFium-backed validation and a PDFKit fallback path.
- User files outside Orifold's app support directories are preserved on uninstall.
