*Released 2026*

Orifold v5 tightens the everyday workflow around saving, exporting, and small editing details.

- **PDF-first save path** — standard macOS Save and Save As write flat PDF output by default; the export menu still covers DOCX, Markdown, text, HTML, image pages, and print. See [Export & save](/Orifold/export/export-save/).
- **Source-aware export** — unchanged same-format exports can return original bytes for supported source documents; `.doc`, `.odt`, and `.rtf` joined the export menu.
- **Inline text editing polish** — refinements to the editing experience introduced in v4. See [Edit existing PDF text](/Orifold/edit/edit-text/).
- A more organized toolbar and an optional in-app helper.
