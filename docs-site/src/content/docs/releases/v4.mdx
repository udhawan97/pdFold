*Released July 1, 2026*

Orifold v0.4 focuses on the editing and signing paths that matter most in dense real-world PDFs.

- **Click existing text and edit it in place**, keeping the original page stable. See [Edit existing PDF text](/Orifold/edit/edit-text/).
- **Digital signatures** — draw and place a signature, export a signed PDF. See [Sign documents](/Orifold/fill-sign/signatures/).
- Hardened PDF save metadata and export integrity.
- Xcode project parity and expanded regression coverage.
