Every exported PDF passes a qpdf structural check before it's written to disk. This catches malformed output before it ever reaches you, rather than leaving you to discover a broken file later.

<Callout variant="note">
	If this check fails, export stops and Orifold reports the problem instead of writing a bad file — see [Export & save problems](/Orifold/help/troubleshooting/export/) for what the message means and what to do next.
</Callout>
