## Steps

1. Press <Kbd>⌘E</Kbd> or choose **File → Export**.
2. Choose a format: PDF, DOCX, Markdown, plain text, HTML, PNG pages, or JPEG pages. Choose Print to send straight to a printer.
3. Pick a location in the save panel and confirm.

<Figure src="/Orifold/assets/gifs/export-save-confirmation.png" alt="The Export dialog: Format set to PDF (.pdf), with Protect with password, Reduce file size, and Sanitize for sharing checkboxes, and Cancel / Export buttons." caption="The Export dialog — pick a format, toggle protection, compression, or sanitize, then Export." />

## Where did my file go?

Exactly where you chose in the save panel — there's no hidden folder or upload step. If you're not sure where you saved it, check the location shown in the save dialog before you clicked Save, or search Spotlight for the filename.

<Callout variant="note">
	**Why it matters:** every exported PDF passes a structural integrity check before it reaches disk — a broken file never gets written. If export fails, see [Export & save problems](/Orifold/help/troubleshooting/export/).
</Callout>

<PetTip who="gami">
	Sharing a final copy? Gami suggests locking form answers and baking stamps at export, so nobody "accidentally" edits your finished document.
</PetTip>

## What you should see

A new file at the location you chose, in the format you picked, reflecting every change made in the workspace.
