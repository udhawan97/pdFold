## Steps

1. Choose Export, then enable the sanitize pass.
2. Optionally include stripping document metadata (author, creation tool, and similar fields).
3. Export as usual — choose the destination, and the sanitize pass then runs with a progress indicator and a working **Cancel**, leaving the window responsive.

## What it strips

Catalog auto-run actions, embedded JavaScript, embedded files, and — opt-in — document metadata.

**Why it matters:** a PDF can quietly carry scripts, attachments, and metadata that travel with the file. Sanitizing means the copy you send contains nothing you didn't intend to send.

## What you should see

The exported PDF opens and displays identically, but carries none of the stripped content.
