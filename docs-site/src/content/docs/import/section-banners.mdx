<Callout variant="whentouse">
	You're combining several documents into one packet and want a clear divider between each source — for exhibits, applications, or client packets.
</Callout>

## Steps

1. In the sidebar, select the point where you want a divider.
2. Add a section banner page — it becomes a labeled break between the documents on either side of it.
3. Rename the banner to describe the section it introduces.

## What you should see

A labeled divider page sits between the two documents in the sidebar and in the exported PDF, making the packet easy to navigate.
