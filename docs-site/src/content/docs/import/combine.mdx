	- You have six PDFs and two screenshots that need to become one attachment.
	- A client sent revisions as separate files that all belong in one packet.
</Callout>

## Steps

1. Import every file you want in the final document (drag them in, or <Kbd>⇧⌘O</Kbd>).
2. In the sidebar, drag pages up or down — you can reorder across documents, not just within one.
3. Delete or rotate any pages you don't need.
4. Export once as a single PDF.

   <Figure src="/Orifold/assets/gifs/combine-reorder-pages.svg" alt="A page dragged from one document into another document's position in the sidebar, with an insertion line marking the drop point." caption="Reordering pages across documents in the sidebar." />

<Callout variant="tip">
	Merging isn't a separate step in Orifold — the workspace *is* the merge. Arrange pages in the sidebar until they tell the story you need, then export once.
</Callout>

## What you should see

One exported PDF, in the page order you arranged, containing content from every file you imported.
