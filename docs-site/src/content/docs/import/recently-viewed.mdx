<Callout variant="whentouse">
	You closed Orifold and want to get back to a file you were working on.
</Callout>

## Steps

1. Launch Orifold with no workspace open.
2. The empty-state screen shows a **Recently Viewed** shelf with thumbnails of the last files you opened.
3. Click a thumbnail to reopen that file where you left off.

<Figure alt="PLACEHOLDER for screenshots/recently-viewed-shelf.png — empty-state screen with the Recently Viewed shelf, 3–4 demo-file thumbnails — dark mode, 1600×1000, 2x" />

<Callout variant="note">
	The shelf and its thumbnails are stored only on your Mac — nothing about your recent files leaves the machine.
</Callout>

## What you should see

The file reopens in the same state you left it, including page order and any edits already made.
