</Callout>

## Steps

1. Open the sidebar page list.
2. **Reorder** — drag a page to a new position, even into a different document's section.
3. **Rotate** — right-click a page and choose a rotation, or use **Rotate Left / Rotate Right** from the **···** menu's Pages section for the page currently on screen.
4. **Delete** — right-click a page and choose delete, or use **Delete Page…** from the same **···** menu.

<Figure src="/Orifold/assets/gifs/reorder-rotate-delete-pages.png" alt="The ··· menu open, showing the Pages section with Rotate Left, Rotate Right, Duplicate Page, and Delete Page actions for the current page." caption="Rotating or deleting the current page from the ··· menu — right-clicking a page thumbnail in the sidebar offers the same actions." />

<Callout variant="tip">
	⌘Z is fearless. Page deletes, rotations, and edits all live in undo history — experiment freely.
</Callout>

## What you should see

The sidebar reflects your new order immediately; the canvas jumps to whichever page you last touched.
