## Steps

1. Import the file as usual.
2. Orifold shows a password prompt — enter the password to unlock it.
3. The file's permissions (printing, editing, and so on) are respected once unlocked.

## What you should see

The file opens into your workspace like any other import, fully editable within the permissions it was granted.
