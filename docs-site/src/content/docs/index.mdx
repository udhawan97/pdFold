## Many files in. One PDF out.

That "simple PDF task" is rarely one file — it's six PDFs, two screenshots, a Word doc, and a scanned form. Orifold takes all of it in, lets you finish it in one place, and hands back a single clean document.

<Figure src="/Orifold/assets/orifold-workspace-diagram.svg" alt="Orifold product flow: import mixed files, edit and finish them in one local workspace, then export one clean file — everything runs on your Mac" />

## What makes it different

<div class="of-columns">
	<div class="of-diff">
		<strong>Local-only</strong>
		<p>Import, OCR, encryption, and export never touch a network.</p>
	</div>
	<div class="of-diff">
		<strong>Merging isn't a step</strong>
		<p>Arrange files in one workspace, then export once.</p>

	<div class="of-diff">
