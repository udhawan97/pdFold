Every Orifold workspace is built from the same four regions.

<Figure src="/Orifold/assets/screenshots/the-orifold-window-annotated.png" alt="The Orifold workspace: the document sidebar on the left, the annotation toolbar and page controls across the top, and the Sample Proposal page filling the canvas." caption="The Orifold window: sidebar (left), toolbar (top), canvas (center). The inspector panel opens from the toolbar on the right when needed." />

## Sidebar

Lists every document and page in the workspace. Drag to reorder pages, right-click to rotate or delete, and use it to jump to any page. The **Recently Viewed** shelf lives here too when no workspace is open.

## Canvas

The page itself — where you read, click to edit detected text, place text boxes, annotate, sign, and fill forms. The page indicator at the bottom shows your position; ⌘F opens workspace-wide search.

## Annotation toolbar

Highlight, underline, strikeout, notes, ink, text boxes, and stamps live here. **Export** is a direct toolbar action. Reader Mode, Contents, Read Aloud, Document Comfort, barcode tools, and archival-readiness guidance are behind **More** (⋯) or in the macOS menus.

## Inspector

Eight focused tabs cover **Info**, **Tags**, **Comments**, **Markup**, **Decorate**, **OCR**, **Attachments**, and **Structure**. Info edits title, author, subject, and keywords; Attachments manages embedded files; Structure shows the current page's tagged reading-order tree without changing it.
