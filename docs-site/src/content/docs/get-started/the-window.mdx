Every Orifold workspace is built from the same four regions.

<Figure src="/Orifold/assets/screenshots/the-orifold-window-annotated.png" alt="The Orifold workspace with the document sidebar on the left, the annotation toolbar across the top, and a page's content filling the canvas." caption="The Orifold window: sidebar (left), toolbar (top), canvas (center). The inspector panel opens from the toolbar on the right when needed." />

## Sidebar

Lists every document and page in the workspace. Drag to reorder pages, right-click to rotate or delete, and use it to jump to any page. The **Recently Viewed** shelf lives here too when no workspace is open.

## Canvas

The page itself — where you read, click to edit detected text, place text boxes, annotate, sign, and fill forms. The page indicator at the bottom shows your position; ⌘F opens workspace-wide search.

## Annotation toolbar

Highlight, underline, strikeout, notes, ink, text boxes, and stamps live here. Reader Mode and Document Comfort's reading presets are a tap away behind the **More** (⋯) button.

## Inspector

Document details, the list of annotations and comments on the current file, and form-field summaries.
