Every Orifold workspace is built from the same four regions.

<Figure src="/Orifold/assets/screenshots/the-orifold-window-annotated.svg" alt="The Orifold workspace with four labeled regions: the sidebar of documents, the annotation toolbar, the page canvas, and the inspector." caption="The Orifold window: sidebar (left), canvas (center), annotation toolbar (top), inspector (right)." />

## Sidebar

Lists every document and page in the workspace. Drag to reorder pages, right-click to rotate or delete, and use it to jump to any page. The **Recently Viewed** shelf lives here too when no workspace is open.

## Canvas

The page itself — where you read, click to edit detected text, place text boxes, annotate, sign, and fill forms. The page indicator at the bottom shows your position; ⌘F opens workspace-wide search.

## Annotation toolbar

Highlight, underline, strikeout, notes, ink, text boxes, and stamps live here, along with the Document Comfort popover for Reader Mode and Night Mode.

## Inspector

Document details, the list of annotations and comments on the current file, and form-field summaries.
