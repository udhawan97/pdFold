## Steps

- **Match formatting** — select a text box, then choose Match Formatting and click nearby PDF text to copy its font, size, and color.
- **Copy formatting** — copy the style from one text box to reuse on another.
- **Paste formatting** — apply a copied style to the selected text box.
- **Reset formatting** — return a text box to Orifold's default style.

## What you should see

The text box's font, size, and color update immediately to match your chosen source.
